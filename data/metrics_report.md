# Capability Retention Metrics Report

This report evaluates the Multi-Agent System (Student) analyzing DE-IDENTIFIED textual data against the fixed Claude Opus 4.7 benchmark (Grader) which evaluated RAW, unmodified data.

## Classification Metrics
- **Total Emails Evaluated**: 2000
- **System Precision**: 2.38%
- **System Recall**: 2.99%
- **F1-Score**: 2.65%
- **Overall Accuracy**: 92.65%

## Error Rates & Agent Behavior
- **False Positive Rate (FPR)**: 4.24% (Clean emails incorrectly flagged as threats)
- **False Negative Rate (FNR)**: 97.01% (True threats that bypassed the system)
- **Deliberation Trigger Rate**: 74.25% (Frequency of Agent Disagreement)

*(Note: Network Topology capabilities were bypassed to isolate text-based threat capability retention.)*

## Confusion Matrix

## Research Insights & Failure Analysis
### 1. The PII-Preservation Trade-off
Our evaluation shows that by completely redacting structural PII (Names, Emails, Phones), the system's ability to detect explicit textual threats resulted in an F1-Score of 2.65%. The 82 False Positives represent instances where the cognitive agents were overly zealous in the absence of precise name context, while the 65 False Negatives represent threats that successfully bypassed the Sentiment Agent due to de-identification destroying referential context. The 74.25% Deliberation Rate highlights how often the removal of context caused the Investigator and Sentiment agents to completely disagree.

### 2. System Prompt Asymmetry
This evaluation tested a live Multi-Agent System equipped strictly with a generalized corporate policy (`sentiment_policy.md`) against a Ground-Truth benchmark generated by an Expert Grader equipped with a highly specific forensic fraud taxonomy (`acfe_enron_taxonomy.md`). The live system's lower recall highlights the extreme difficulty of zero-shot generalized threat detection compared to taxonomy-driven analysis.

## Future Research Directions
1. **Context-Aware De-identification:** Instead of scrubbing `[PERSON_A]`, map individuals to synthetic but consistent personas (e.g., `VP_Finance_1`) to preserve network topology for the Investigator Agent without leaking real PII.
2. **Dynamic Taxonomy Retrieval:** Allow the Deliberation Agent to fetch specific ACFE fraud taxonomies dynamically via RAG when initial generic policies are triggered.
3. **Keep the Raw Baseline:** Yes, the RAW Ground Truth must remain the immutable baseline. You can only measure the "privacy cost" of an anonymization technique by comparing its performance against the unimpeded, raw data performance baseline.
